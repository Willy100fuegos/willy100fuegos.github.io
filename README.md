## 📄 Sobre este Proyecto

Este repositorio aloja el código fuente (`index.html`) de mi portafolio personal. A diferencia de un CV tradicional, este sitio está diseñado como un **Showcase Interactivo** que redirigir a los visitantes a las demostraciones técnicas de mis otros repositorios (C4, TQPM, CoatzaSeguro).

🔗 **Ver sitio en vivo:** [https://willy100fuegos.github.io/](https://willy100fuegos.github.io/)

---

## 🛠️ Stack Tecnológico (Frontend)

El sitio ha sido construido bajo una arquitectura **Serverless / Static** para máxima velocidad de carga y cero mantenimiento.

| Tecnología | Uso |
| :--- | :--- |
| **HTML5 Semántico** | Estructura del contenido y accesibilidad. |
| **Tailwind CSS (CDN)** | Estilizado rápido utilitario (`script src`). Diseño responsivo "Mobile-First". |
| **FontAwesome 6** | Iconografía vectorial para las tarjetas de proyectos. |
| **GitHub Pages** | Hosting estático y despliegue continuo (CD). |
